# IMAGES-PROMPTS-V2.md — Jardins de Prestige

> Guide de generation d'images IA pour le site vitrine paysagiste haut de gamme.
> Entreprise : Jardins de Prestige — Lyon, France.
> Toutes les images doivent refleter un positionnement premium, un savoir-faire francais et une esthetique contemporaine epuree.

---

## Instructions generales pour la generation

- Style photographique : reportage architectural haut de gamme, lumiere naturelle
- Palette : verts profonds, pierre claire, bois chaud, touches d'or et de gris anthracite
- Aucun filigrane, aucun texte dans l'image
- Qualite : hyperrealiste, prise de vue professionnelle, DSLR full-frame
- Mood : luxe discret, soin du detail, nature maitrisee
